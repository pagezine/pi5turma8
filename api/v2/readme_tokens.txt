    id: Um campo de tipo inteiro que é a chave primária e auto-incrementável, o que significa que ele será incrementado automaticamente para cada novo registro.
    token: Um campo de tipo VARCHAR(255) para armazenar os tokens.
    status: Um campo de tipo TINYINT(1) que pode ser usado para definir o status do token. Neste exemplo, status é um campo opcional e é definido como 1 por padrão.
